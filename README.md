#### THE GOLOCAL ###
GoLocal is an intuitive, real-time command center designed to make city transit easier for everyone. By combining live vehicle tracking, instant incident reporting, and quick-access ticketing into one simple, human-centered interface, GoLocal transforms urban mobility into a predictable and frictionless experience.

### The Vision ###
Urban transit is often plagued by fragmented data and slow response times. GoLocal bridges this gap by providing a unified, real-time dashboard that synchronizes urban transit for smarter, faster, and more reliable travel. Whether for commuters or dispatchers, GoLocal provides the clear, actionable insights needed to eliminate the guesswork from getting around.


### Key Features
Real-Time Fleet Intelligence: Monitor active routes, crowd levels, and vehicle speeds using the interactive "Transit Route Cartographer."
Proactive Incident Management: Quickly identify and resolve disruptions like traffic congestion or flooding through a centralized, real-time ledger.
Frictionless Booking: A simplified, conversational QR booking flow allows passengers to secure their journey in seconds.
AI-Powered Coordination: An integrated AI companion provides instant updates on wait times, delays, and fare details, acting as the intelligent core of the transit system.


 ### Tech Stack
Frontend: React, TypeScript
Runtime/Tooling: Node.js, tsx
Real-time Capabilities: Simulated live telemetry and incident monitoring


### Getting Started
Prerequisites
Ensure you have Node.js installed on your machine.
Installation
Clone the repository and install the required dependencies:-
npm install


### Running the Project
To start the development server, run:
npm run dev
The application will be accessible via the local port specified in your terminal.
